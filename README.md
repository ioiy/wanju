# Wanju (哪吒探针脚本修改版)

本项目是基于“老王大佬”的脚本进行的修改版本。主要解决的问题是原脚本无法正确下载能显示国旗的 **v0 版本哪吒探针**。本仓库的脚本经过简单修改，增加了自动识别 **v0** 或 **v1** 版本哪吒探针的功能，并进行适配部署。

## 📂 目录结构与使用说明

本仓库包含针对不同环境（Java, Node.js, PHP, Teamspeak3, SAP 等）的适配脚本。

### 1. Java 环境 (`java`)

适用于 Java 环境的部署方案。

* **文件说明**：
    * `server.jar`: 启动文件（可根据实际情况修改文件名）。
    * `start.sh`: 配置文件，请在此文件中修改具体的配置参数。
    * `LICENSE.jar`: 伪装的游戏系统文件。**注意**：部分游戏机/服务器如果没有检测到该游戏系统会自动关机，请保留此文件。
* **功能特性**：
    * 自动识别并下载适配的 v0 (带国旗显示) 或 v1 哪吒探针。

### 2. Teamspeak3 Server (`Teamspeak3-server`)

适用于 Teamspeak3 服务器的部署方案。

* **部署步骤**：
    1.  **备份/重命名**：上传文件前，请先将服务器上原有的 `ts3server` 游戏文件重命名为 `LICENSE`（如果已存在 `LICENSE` 文件可先删除，但**不要删除其他文件**）。
    2.  **上传**：上传本目录下的压缩包并解压，得到新的 `ts3server` 文件。
    3.  **配置**：双击 `ts3server` 文件添加固定隧道或哪吒配置。
    4.  **保存**：保存时请选择 **shell** 格式保存（通常在保存按钮的左边）。
    5.  **权限**：给予文件 `777` 权限。
    6.  **启动**：点击 `start` 开机。
* **节点信息**：
    * 节点信息将保存在 `tsdns` 文件夹里的 `sub.txt` 文件中。
* **功能特性**：
    * 集成自动识别 v0/v1 哪吒探针的功能。

### 3. Node.js 环境 (`nodejs` / `Webhostmost-ws-nodejs`)

适用于 Node.js 环境的部署方案。

* **文件说明**：
    * `index.js`: 启动文件（可根据实际情况修改文件名）。
    * `start.sh`: 配置文件，请在此文件中修改配置。
* **功能特性**：
    * 自动识别并下载适配的 v0 (带国旗显示) 或 v1 哪吒探针。

### 4. PHP 环境 (`php`) [🆕 新增]

适用于 PHP 虚拟主机或容器环境的部署方案。

* **文件说明**：
    * `index.php`: 入口文件，用于在 Web 访问时触发/维持进程。
    * `start.sh`: 配置文件，**请务必在此文件中修改具体的参数（如面板地址、Key 等）**。
* **部署说明**：
    * 将文件上传至网站根目录（通常为 `public_html` 或 `wwwroot`）。
    * 确保 `start.sh` 和 `index.php` 具有可执行权限（如 `755` 或 `777`）。
    * 访问您的网站首页即可激活探针。
* **功能特性**：
    * 完美支持 v0 和 v1 哪吒探针的自动识别与下载。

### 5. SAP (`SAP`)

适用于 SAP 环境的脚本。

* **注意事项**：
    * **请记得混淆代码后再进行部署**。
* **功能特性**：
    * 同样支持 v0 和 v1 哪吒探针的自动识别与下载。

## 🛠️ 关于修改

原脚本在下载哪吒探针时可能存在版本兼容问题，导致无法显示国旗（v0版本特性）。本仓库通过修改判断逻辑，使其能够根据需求自动识别并下载正确的二进制文件：

* **v0**: 支持国旗显示。
* **v1**: 最新版探针。
